Website Mentari Sehat Indonesia Kab Demak

### Requirements :
- Composer v2
- PHP v8.2
- Laravel v11.0


### Instalations

Clone Github
```
git clone https://github.com/hasbiardianto/web-msi-demak.git
```

Update Composer
```
composer update
```

Install npm
```
npm install
```

Migrate Database
```
php artisan migrate
```
